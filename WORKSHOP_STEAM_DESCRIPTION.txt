[h1]Online Time Trials[/h1]

[b]What it is[/b]
Place timed trial courses in any mission (SP/MP): ordered segment waypoints (gates, hovers, sling loads, destroy targets, and more), HUD timing, and optional leaderboards.

[b]Requirements[/b]
[list]
[*][url=https://steamcommunity.com/workshop/filedetails/?id=450814997]CBA_A3[/url] — default keybind to open the trial menu.
[/list]

[img]https://raw.githubusercontent.com/Goltred/arma3-glt-online-time-trials/refs/heads/main/steam/Hero.png[/img]

[h2]Quick start (mission makers)[/h2]
[list]
[*]Eden: [b]Empty → Online Time Trials[/b] — place one [b]Trial Definition[/b] ([i]GLT_Trials_TrialMeta[/i]) per course; set [b]Trial Id[/b] (unique) and [b]Trial Name[/b].
[*]Place segment modules; set [b]Segment Index[/b] ascending; [b]sync each segment to the Trial Definition[/b].
[*]Optional: [b]Master[/b] (leaderboard persistence), [b]Terminal[/b] (in-game leaderboard screen).
[/list]
Trial Definition objects are editor helpers and are removed at mission start — that is normal.

[h2]Players[/h2]
[list]
[*]Get in an [b]eligible vehicle[/b] as [b]driver/pilot[/b].
[*][b]Shift + T[/b] (CBA: “Time Trials - Select Trial”) → pick trial → [b]OK[/b].
[*]Near a [b]Terminal[/b]: scroll action [b]Time Trials Terminal[/b] for live runs / leaderboard.
[/list]
